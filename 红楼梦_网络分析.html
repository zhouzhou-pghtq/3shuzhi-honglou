8.1 项目背景与数据准备

节点数据集 (nodes.csv) 构建要素： Id (整数) / Label (人物) / Gender (性别) / Faction (阵营)
边数据集 (edges.csv) 构建要素： Source (源Id) / Target (目标Id) / Weight (互动权重) / Type (Undirected 无向)

人物选取（20位）： 贾宝玉(1)、林黛玉(2)、薛宝钗(3)、王熙凤(4)、贾母(5)、贾政(6)、王夫人(7)、探春(8)、史湘云(9)、妙玉(10)、李纨(11)、贾琏(12)、贾珍(13)、袭人(14)、晴雯(15)、平儿(16)、刘姥姥(17)、薛蟠(18)、元春(19)、秦可卿(20)。

8.4 步骤四：社区发现 (Community Detection) - Louvain 算法模拟

通过在 Gephi 中运行 Louvain 模块度算法（Resolution=1.0），网络被自动分割为 4 个核心社群（Modularity Class）。

🔴 社区 1：贾府权力与统治中枢（成员：贾母、王熙凤、贾政、王夫人、贾琏、贾珍、元春）
特征：掌握贾府财政、人事任免和宗族大权的阶层。王熙凤是这里的“超级中介”，上承贾母意志，下压平儿、仆役。元春虽身在宫中，却通过省亲事件与此社区深度绑定。

🔵 社区 2：大观园审美与情感净土（成员：贾宝玉、林黛玉、薛宝钗、探春、史湘云、妙玉、李纨）
特征：由青春、诗意和自由意志主导的“乌托邦”社区。受封建礼法束缚最小，但极度依赖“社区1”的经济供养。贾宝玉是唯一能将这份情感向外部传递的脆弱通道。

⚫ 社区 3：奴仆与底层依附者（成员：袭人、晴雯、平儿、秦可卿）
特征：处于权力网络的最底层。平儿周旋于王熙凤与贾琏之间，袭人和晴雯是宝玉生活上的左膀右臂。她们是宏大封建叙事中的“齿轮”和牺牲品。

🟢 社区 4：外部冲击与世情边缘（成员：刘姥姥、薛蟠）
特征：与贾府核心有极强互动的“外人”。刘姥姥的“进大观园”是打破阶层壁垒的特殊节点，薛蟠则是四大家族外部经济圈的代表。

8.5 步骤五：结果分析与批判性阐释

数据印证了《红楼梦》“千红一哭，万艳同悲”的家族必然性。

从“度中心性”和“中介中心性”的量化结果来看，王熙凤不仅是封建大家庭中最忙碌的人，更是贾府这艘大船得以维系运转的“舵手”。她超高的“中介中心性”意味着她是“主子们”与“下人们”之间的唯一桥梁。但她的权力核心地位，恰恰也是贾府内部腐败和裂缝的根源。

贾宝玉作为“大观园女儿国”情感的代言人，他极高地承担起了连接“社区1”和“社区2”的重任。遗憾的是，这个连接者本身是软弱无力的。当“社区1”（贾政、王夫人等）为了家族利益强行拆解“社区2”（逼婚、抄检大观园）时，宝玉的桥梁作用瞬间断裂。这直接导致了大观园网络结构的崩溃。

至于林黛玉，她的度中心性（7）并不算极高，但结合原著来看，她的精神领袖地位是通过“情感共鸣”而非“社交广博”建立的。她连接的是社区2中灵魂深处的孤独（湘云、妙玉）。当这批“精神岛屿”无法融入“权力海洋”时，她们的结局只能是“冷月葬花魂”。

最终，当王熙凤的权力失控、贾母的权威老去、大观园被封建礼教吞噬，这个高度紧凑的家族网络便不可避免地走向了“白茫茫大地真干净”的悲剧终局。
